# Claude Code Boricua Spinner Verbs

Add some sazón to your Claude Code sessions with loading spinner verbs from the island.

## Setup

Copy the `spinnerVerbs` block from `spinner-verbs.json` into your Claude Code `settings.json`:

- **Global:** `~/.claude/settings.json`
- **Project:** `.claude/settings.json`

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": ["Bregando", "Chambeando", "..."]
  }
}
```

Use `"mode": "append"` to keep the defaults and add these on top.
